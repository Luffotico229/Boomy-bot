<h2 align="center">⚡ Boomy Bot — Cyberpunk Edition</h2>

Boomy Bot is a Discord bot I am developing as a personal project, inspired by the **Cyberpunk Edgerunners** aesthetic.  
Its main goal is to provide clean, fast and stylish utility commands — including automatic role assignment and user information scans.

---

## 🚀 Current Features

### `/bmy setautorol`
Configures an existing server role to be automatically assigned to every new member.  
The bot stores the configuration per server using a lightweight `autoroles.json` file.

### `/bmy avatar`
Displays the avatar of a user by mention or ID.

### `/bmy banner`
Shows the profile banner of a user (if they have one).

### `/bmy info`
Performs a **Lucy-style netrunner scan**, returning:

- Avatar  
- Banner  
- User ID  
- Account creation date  
- Server join date  
- Roles  
- All inside a themed cyberpunk embed  

---

## 🛠️ Technologies Used
- Python  
- discord.py (app_commands)  
- JSON for lightweight storage  
- Railway for 24/7 hosting
