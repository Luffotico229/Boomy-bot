# Boomy Bot

Boomy is a Discord bot that I am developing as a personal project.  
Right now, its main feature is the ability to set an **automatic role** for new members using a slash command.

## Current Features

### `/setautorol`
Allows you to select an existing role in the server.  
That role will be automatically assigned to every new member who joins.

The bot stores the configuration per server using a simple `autoroles.json` file.

## Technologies Used
- Python
- discord.py (app_commands)
- JSON for lightweight storage
- Railway for 24/7 hosting
